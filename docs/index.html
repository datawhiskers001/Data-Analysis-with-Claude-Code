1. Install Visual Studio Code
A free app where you write and read code — made by Microsoft
💡 Think of VS Code like Microsoft Word, but for writing code. It's free and safe to download from Microsoft's official website.
I have a Mac
I have Windows
1

Go to the VS Code website and click Download

Click the link below — it will open the official Microsoft page. Look for the big blue button that says "Download for Mac" and click it. A file will start downloading.

🌐 Click here to go to the VS Code download page
2

Find the downloaded file and unzip it

Open your Downloads folder (you can find it in the Dock at the bottom of your screen, or open Finder and look in the left sidebar). You'll see a file called something like VSCode-darwin-universal.zip. Double-click it — it will unzip into an app icon.

3

Move VS Code to your Applications folder

You'll now see the VS Code icon (it looks like a blue "<>" symbol). Drag it into your Applications folder — this is just like installing any other Mac app. If you're not sure how, just right-click the icon and choose Open from the Downloads folder.

4

Open VS Code for the first time

Double-click the VS Code icon in your Applications folder. Your Mac might ask "Are you sure you want to open this?" — just click Open. That's a normal safety check for downloaded apps. VS Code will open and you'll see a welcome screen.

✅ Great job! VS Code is installed. You'll see a welcome screen with some options — you can ignore those for now and move on to the next step.
1

Go to the VS Code website and click Download

Click the link below — it opens the official Microsoft page. Look for the big blue button that says "Download for Windows" and click it. A file ending in .exe will download to your computer.

🌐 Click here to go to the VS Code download page
2

Run the installer

Go to your Downloads folder and double-click the file you just downloaded (it's named something like VSCodeSetup-x64-1.xx.x.exe). Windows may ask "Do you want to allow this app to make changes?" — click Yes.

3

Follow the setup wizard — just keep clicking Next

An installer window will appear. You can leave everything as the default. On the screen titled "Select Additional Tasks", make sure these two boxes are checked — they make VS Code easier to use later:

  • ☑ Add "Open with Code" action to Windows Explorer file context menu — lets you right-click any file and open it in VS Code
  • ☑ Add to PATH — lets you open VS Code from the command line later (needed for Claude Code)
4

Click Install, then Finish

Click the Install button and wait about 30 seconds for the progress bar to finish. Then click Finish. VS Code will open automatically — you'll see a welcome screen.

✅ VS Code is installed! You'll see a welcome screen — you can ignore it for now and move on to the next step.

3. Install Claude Code
The tool that lets Claude AI help you directly inside your projects
💡 Claude Code runs in a Terminal (a text-based window where you type commands). Don't worry — you'll just copy and paste the commands shown below. This section also requires installing Node.js first, which is a free helper program that Claude Code needs to run.
I have a Mac
I have Windows
1

Download and install Node.js

Node.js is a free program that Claude Code needs in order to run — think of it as a required background engine. Click the link below, then click the big "Download Node.js (LTS)" button. Open the downloaded .pkg file and follow the simple on-screen steps (just keep clicking Continue → Install → Close).

🌐 Click here to download Node.js
2

Open the Terminal app on your Mac

The Terminal is a black (or white) text window where you type commands. To open it: press ⌘ Space to open Spotlight search, type Terminal, and press Enter. A window will appear — don't be alarmed by the plain look, it's perfectly normal!

3

Install Claude Code — just copy and paste this command

Click the Copy button on the right, then click inside the Terminal window and paste it (⌘ V), and press Enter. You'll see some text scrolling — that's normal, it means it's working. Wait until you see a new line with a $ or % symbol, which means it's done.

Terminal — paste this and press Enter 
npm install -g @anthropic-ai/claude-code
4

Start Claude Code and log in with your Claude account

Type the command below and press Enter. Claude Code will automatically open your browser and ask you to sign in — use the same email and password you created your Claude Pro account with. Once you approve it in the browser, come back to Terminal.

Terminal — paste this and press Enter 
claude
ℹ️ Your browser will open a Claude page saying something like "Allow Claude Code to access your account?" — click Approve or Allow. Then go back to Terminal where Claude Code will be ready to chat.
5

Open Claude Code inside VS Code (the easiest way to use it)

Open VS Code. At the top menu click Terminal → New Terminal. A terminal panel will appear at the bottom of VS Code. Type claude and press Enter — Claude Code will start right inside your editor, ready to help with your files.

✅ You're all set! Claude Code is installed and connected to your account. You can now open any project folder in VS Code, open the terminal, type claude, and ask Claude to help you with your work.
1

Download and install Node.js

Node.js is a free helper program that Claude Code needs to run. Click the link below, then click the big green button that says "Download Node.js (LTS)". Open the downloaded .msi file, click Next a few times, then Install, then Finish — just like installing any other Windows program.

🌐 Click here to download Node.js
2

Open PowerShell as an Administrator

PowerShell is a text window where you type commands (similar to Terminal on a Mac). To open it: click the Start menu (Windows logo), type PowerShell, then right-click on Windows PowerShell and choose "Run as administrator". Click Yes when Windows asks for permission.

⚠️ It's important to run it "as administrator" — without this, the install may fail with a permissions error.
3

Install Claude Code — copy and paste this command

Click the Copy button, then click inside the PowerShell window, right-click to paste, and press Enter. You'll see text scrolling — that's normal. Wait until a new line appears with a > symbol at the start.

PowerShell — paste this and press Enter 
npm install -g @anthropic-ai/claude-code
4

If you see a "script cannot be loaded" error — run this fix

Some Windows computers block this by default. If you see an error mentioning "scripts" or "execution policy", copy and paste the command below and press Enter. This is a safe, standard Windows setting.

PowerShell — only run this if you got an error above 
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

After running that, go back and repeat Step 3.

5

Start Claude Code and log in with your Claude account

Type the command below in PowerShell and press Enter. Your browser will open automatically asking you to sign in to Claude — use the same email and password from your Claude Pro account. Click Allow or Approve, then come back to PowerShell.

PowerShell — paste this and press Enter 
claude
ℹ️ Once you approve in the browser, go back to PowerShell — you'll see Claude's prompt waiting for you. You're connected!
6

Open Claude Code inside VS Code (the easiest way to use it)

Open VS Code. At the top menu click Terminal → New Terminal. A terminal panel will appear at the bottom. Type claude and press Enter — Claude Code will start inside VS Code, ready to help you with your files.

✅ You're all set! Claude Code is installed and connected to your account. Open any project folder in VS Code, open the terminal at the bottom, type claude, and start asking Claude to help with your work.
